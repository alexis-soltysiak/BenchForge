Write a Python function `two_sum(nums, target)` that returns the indices of two numbers whose sum equals `target`. Provide an efficient solution.

gpt-5.4-mini-2026-03-17

```python
def two_sum(nums, target):
    seen = {}  # number -> index

    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

    return []  # if no solution exists
```

### How it works
- Use a dictionary to store numbers we've already seen and their indices.
- For each number `num`, check whether `target - num` has already appeared.
- If yes, return the two indices immediately.

### Efficiency
- **Time complexity:** `O(n)`
- **Space complexity:** `O(n)`

### Example
```python
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
```
