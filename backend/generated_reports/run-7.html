```python
def two_sum(nums, target):
    seen = {}  # value -> index

    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

    return None  # or raise an exception if no solution exists
```

### How it works
- As you scan the list once, keep a dictionary of numbers you've already seen.
- For each number `num`, check whether `target - num` was seen before.
- If yes, return the stored index and the current index.

### Efficiency
- **Time:** `O(n)`
- **Space:** `O(n)`

### Example
```python
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
```
